# shreyasgaonkar.com

Personal portfolio site for Shreyas Gaonkar - Systems Development Engineer at AWS, scaling Amazon CloudWatch Logs at 20+ Tbps.

[shreyasgaonkar.com](https://shreyasgaonkar.com)

Built with vanilla HTML, SCSS, and JavaScript. No frameworks.

## Setup

```bash
npm install
```

## Build

```bash
npm run build
```

## Dev (watch + live reload)

```bash
npm run dev
```

Opens `localhost:3000` with auto-reload on file changes.

## Content

Update sections by editing the JSON files in `data/`:

- `experience.json` - Work history
- `projects.json` - Relevant projects
- `articles.json` - Published AWS articles
- `testimonials.json` - Testimonials
